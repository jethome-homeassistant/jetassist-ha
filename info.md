## JetAssist

Connect your Home Assistant to JetAssist for remote access, cloud backups,
and AI-powered voice assistant.

### Features

- Secure remote access via WebSocket tunnel
- E2E encrypted cloud backups
- AI assistant with smart home context (coming soon)

### Requirements

- Home Assistant 2025.1.0 or newer
- JetAssist account ([register](https://auth.jethome.cloud))
